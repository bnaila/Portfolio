Projet de d'analyse de données massives

Description

Les données utilisées dans le projet à rendre, qui sont l'ensemble des transactions sur les horodateurs dans la ville de Paris sur l'année 2014. C Celles-ci proviennent du site Open Data Paris, répertoire des données ouvertes de la ville de Paris. Elles sont stockées sur le serveur MongoDB déjà utilisé, dans la base horodateurs.

Elle contient deux collections principales :
  • transactions : ensemble des paiements
  • mobiliers: liste de tous les horodateurs
Exploration des données et proposer une analyse des données horodateurs, en choississant l'angle d'analyse. Elements analysés :
  • Nombres de transactions par arrondissement.
  • Montant total ou moyen par arrondissement.
  • Cartes multiples de type small mutiples.
    • répartition des différents types de mobilier idem pour les types d'alimentation
  • Evolution du montant total des transactions au cours de l'année.
    • globalement par mobilier par arrondissement
  • Regroupement des mobiliers dans des classes.
